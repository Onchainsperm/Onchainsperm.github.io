SPERMO TOKEN

It has a multi-functional role within the ONCHAIN SPERM ecosystem. Not only does it function as a governance token and a reward token for ONCHAIN SPERM NFT holders, but it also serves as a utility token within the ecosystem. This means that users can use SPERMO TOKEN to access various features and services provided by ONCHAIN SPERM, such as transaction fees and other network-related services.
